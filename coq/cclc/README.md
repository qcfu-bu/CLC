# Concurrent Calculus of Linear Constructions

## Core typing rules

```
Γ ▹ U
---------------
Γ ⊢ s@i : U@i+1


Γ ▹ U    Γ ⊢ A : s@i    [Γ, x :s A] ⊢ B : r@i
----------------------------------------------
Γ ⊢ Πt (x :s A).B : t@i


Γ ▹ U
-----------------
Γ, x :s A ⊢ x : A


Γ ▹ t     [Γ] ⊢ Πt (x :s A).B : r@i     Γ, x :s A ⊢ n : B
----------------------------------------------------------
Γ ⊢ λt (x :s A).n : Πt (x :s A).B


Γ2 ▹ s    Γ1 ⊢ m : Πt (x :s A).B     Γ2 ⊢ n : A
------------------------------------------------
Γ1 + Γ2 ⊢ m n : B[n/x]


Γ ⊢ m : A     [Γ] ⊢ B : s@i     A ⪯ B
-------------------------------------
Γ ⊢ m : B
```
