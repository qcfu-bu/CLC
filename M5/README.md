# M5
Combining features of LNLD and QTT.

## Typing Rules

### Rig

* 0: static
* 1: linear
* ω: intuitionistic

```
ρ+0 = 0+ρ = ρ
1+1 = ω
ρ+ω = ω+ρ = ω

ρ*0 = 0*ρ = 0
ρ*1 = 1*ρ = ρ
ω*ω = ω

0 ≤ 0, 0 ≤ &, 0 ≤ ω
1 ≤ 1, 1 ≤ &, 1 ≤ ω 
& ≤ &, & ≤ ω
ω ≤ ω
```

### Axioms
```
----------------   -----------------   ----------------
∅ ⊢ Uᵢʷ : Uᵢ₊₁ʷ     ∅ ⊢ Uᵢ¹ : Uᵢ₊₁ʷ     Γ,x:X¹ ⊢ x : X

Γ₁ ⊢ a : B    Γ₂ ⊢ A : Uᵢʳ
-------------------------
Γ₁,x:A⁰ ⊢ a : B
```

### Type Formation
```
Γ₁ ⊢ X : Uᵢᵃ   Γ₂,x:Xʳ ⊢ Y : Uᵢᵇ   a=1 <=> r=0
----------------------------------------------
Γ₁ + Γ₂ ⊢ (x:Xᵃ) -> Y : Uᵢʷ


Γ₁ ⊢ X : Uᵢᵃ   Γ₂,x:Xʳ ⊢ Y : Uᵢᵇ   a=1 <=> r=0
----------------------------------------------
Γ₁ + Γ₂ ⊢ (x:Xᵃ) >> Y : Uᵢ¹


Γ₁ ⊢ A : Uᵢᵃ   Γ₂,x:Aʳ ⊢ B : Uᵢᵇ   a=1 <=> r=0
----------------------------------------------
Γ₁ + Γ₂ ⊢ Σx:X.Y : Uᵢᶜ    (c = min(a, b))


Γ₁ ⊢ X : Uᵢʷ   Γ₂ ⊢ e : X    Γ₃ ⊢ e' : X
-----------------------------------------
Γ₁ + Γ₂ + Γ₃ ⊢ e ≡ₓ e' : Uᵢʷ
```

### Term Formation
```
Γ₁ ⊢ (x:Xᵃ) -> Y : Uᵢʷ    Γ₂,x:Xʳ ⊢ e : Y    r ≤ a
--------------------------------------------------
Γ₂ ⊢ λx.e : (x:Xᵃ) -> Y


Γ₁ ⊢ (x:Xᵃ) -o Y : Uᵢ¹    Γ₂,x:Xʳ ⊢ e : Y    r ≤ a
--------------------------------------------------
Γ₂ ⊢ λ̂x.e : (x:Xᵃ) -o Y


Γ₁ ⊢ e : (x:Xᵃ) -> Y    Γ₂ ⊢ e' : X
------------------------------------
Γ₁ + aΔ₂ ⊢ e e' : [e'/x]Y


Γ₁ ⊢ e : (x:Xᵃ) -o Y    Γ₂ ⊢ e' : X
------------------------------------
Γ₁ + aΔ₂ ⊢ e e' : [e'/x]Y


Γ₁ ⊢ Σx:Xᵃ.Yᵇ : Uᵢᶜ    Γ₂ ⊢ e : X    Γ₃ ⊢ e' : [e/x]Y
------------------------------------------------------
Γ₂ + Γ₃ ⊢ (e, e') : Σx:Xᵃ.Yᵇ


Γ₁ ⊢ e : Σx:Xᵃ.Yᵇ   Γ₂,x:Xʳ,y:Yˢ ⊢ e' : Z   r ≤ a ∧ s ≤ b
----------------------------------------------------------
Γ₁ + Γ₂ ⊢ let (x, y) = e in e' : let (x, y) = e in Z


Γ₁ ⊢ X : Uᵢʷ    Γ₂ ⊢ x : X
---------------------------
Γ₂ ⊢ reflₓ x : x ≡ₓ x


Γ₁ ⊢ X : Uᵢᵃ    Γ₂ ⊢ e : X     Γ₃,x:Xʳ ⊢ e' : Y    r ≤ a
---------------------------------------------------------
Γ₂ + Γ₃ ⊢ let x = e in e' : [e/x]Y
```
