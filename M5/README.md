Combining features of LNLD and QTT.

## Typing Rules

### Rig

* 0: static
* 1: linear
* ω: intuitionistic

```
ρ+0 = 0+ρ = ρ
1+1 = ω
ρ+ω = ω+ρ = ω

ρ*0 = 0*ρ = 0
ρ*1 = 1*ρ = ρ
ω*ω = ω

0 ≤ 0, 0 ≤ &, 0 ≤ ω
1 ≤ 1, 1 ≤ &, 1 ≤ ω 
& ≤ &, & ≤ ω
ω ≤ ω
```

### Axioms
```
-------------------    --------------------    ------------------------
∅ ; ∅ ⊢ Uᵢʷ : Uᵢ₊₁ʷ    ∅ ; ∅ ⊢ Uᵢ¹ : Uᵢ₊₁ʷ     Γ,x:X ; Γ,x:X¹ ⊢ x : X

Γ ; Δ₁ ⊢ a : B    Γ ; Δ₂ ⊢ A : Uᵢʳ
-----------------------------------
     Γ,x:A ; Δ₁,x:A⁰ ⊢ a : B
```

### Type Formation
```
Γ ; Δ₁ ⊢ X : Uᵢᵃ   Γ,x:X ; Δ₂,x:Xʳ ⊢ Y : Uᵢᵇ    a=1 <=> r=0
------------------------------------------------------------
               Γ ; Δ₁ + Δ₂ ⊢ Πx:Xᵃ.Y : Uᵢᵇ

Γ ; Δ₁ ⊢ A : Uᵢᵃ   Γ,x:A ; Δ₂,x:Aʳ ⊢ B : Uᵢᵇ    a=1 <=> r=0
------------------------------------------------------------
        Γ ; Δ₁ + Δ₂ ⊢ Σx:Xᵃ.Yᵇ: Uᵢᶜ   (where c = min(a, b))

Γ ; Δ₁ ⊢ X : Uᵢʷ   Γ ; Δ₂ ⊢ e : X    Γ ; Δ₃ ⊢ e' : X
-----------------------------------------------------
          Γ ; Δ₁ + Δ₂ + Δ₃ ⊢ e ≡ₓ e' : Uᵢʷ
```

### Term Formation
```
Γ ; Δ₁ ⊢ Πx:Xᵃ.Y : Uᵢᵇ    Γ,x:X ; Δ₂,x:Xʳ ⊢ e : Y    r ≤ a
------------------------------------------------------------
                Γ ; Δ₂ ⊢ λx.e : Πx:Xᵃ.Y

Γ ; Δ₁ ⊢ e : Πx:Xᵃ.Y    Γ ; Δ₂ ⊢ e' : X
----------------------------------------
     Γ ; Δ₁ + aΔ₂ ⊢ e e' : [e'/x]Y

Γ ; Δ₁ ⊢ e : Σx:Xᵃ.Yᵇ   Γ,x:X,y:Y ; Δ₂,x:Xʳ,y:Yˢ ⊢ e' : Z   r ≤ a ∧ s ≤ b
--------------------------------------------------------------------------
     Γ ; Δ₁ + Δ₂ ⊢ let (x, y) = e in e' : let (x, y) = e in Z

Γ ; Δ₁ ⊢ X : Uᵢʷ    Γ ; Δ₂ ⊢ x : X
-----------------------------------
     Γ ; Δ₂ ⊢ reflₓ x : x ≡ₓ x

Γ ; Δ₁ ⊢ X : Uᵢᵃ    Γ ; Δ₂ ⊢ e : X     Γ,x:X ; Δ₃,x:Xʳ ⊢ e' : Y    r ≤ a
-------------------------------------------------------------------------
               Γ ; Δ₂ + Δ₃ ⊢ let x = e in e' : [e/x]Y
```
